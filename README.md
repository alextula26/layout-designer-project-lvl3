### Installing make in Ubuntu

```sh
$ sudo apt update
$ sudo apt install make

```

### Global Installing of npm packages

```sh
$ npm install -g sass
$ npm install pug-cli -g
$ npm install --global gulp-cli
$ npm install -g pug-lint
```

## Setup

```sh
$ make install
```

## Run lint

```sh
$ make lint
```

## Run build using make

```sh
$ make build
```

## Run build using gulp

```sh
$ gulp
```
